# SQL Infrastructure

Scripts SQL para Supabase: schema, RLS y seed.

## Uso

1. Ejecutar `schema.sql` para crear tablas
2. Ejecutar `rls.sql` para configurar Row Level Security
3. Ejecutar `seed.sql` para datos de prueba

## Notas

- RLS debe ajustarse según sistema de autenticación
- Zona horaria: America/Asuncion
- Moneda: PYG

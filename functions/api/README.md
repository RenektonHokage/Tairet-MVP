# API Backend

Express + TypeScript backend para Tairet MVP.

## Endpoints

- POST /orders
- POST /payments/callback
- GET /orders/:id
- PATCH /orders/:id/use
- POST /reservations
- GET /locals/:id/reservations
- GET /locals/:id/promos
- POST /locals/:id/promos
- POST /events/whatsapp_click
- GET /metrics/summary

## Desarrollo

```bash
pnpm dev
```

Ver `.env.example`.
